# SIMPLE BLOG - BOOTSTRAP THEME

A clean and minimalistic blog template.

Includes 3 pages:
- Home / Posts
- Post Details
- Contact

Dependencies
- Bootstrap 3.0.0
- Readable Theme
- Google Fonts
- jQuery 2.0.3
- Respond
- html5shiv
- Font Awesome
